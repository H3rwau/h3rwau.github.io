# 基础作业：

### 1什么是函数的声明？什么是函数的定义？

**函数声明只是对编译系统的一个说明，是对定义的函数的返回值的类型说明，以通知系统在本函数中所调用的函数是什么类型。函数声明是一个说明语句，必须以分号结束。**

**函数的定义是一个完整的函数单元（用来实现函数的功能）**，包含函数类型、函数名、形参及形参类型、函数体等。在程序中，函数的定义只能有一次。

### 2 C语言编写的应用程序从什么函数开始执行的？

从main函数开始，一个程序只能有一个main函数，是整个程序的入口

### 3头文件的里面存放函数声明。请问，在编译过程的哪个步骤里面，这些函数声明会放入代码文件中？

在编译过程的第一个阶段也就是**预处理步骤，会将函数声明一般包括在头文件的内容添加到.cpp文件**。

### 4 C程序是由函数组成的。请问，在编译过程的哪个步骤里面，这些函数会组合成一个应用程序？

由**汇编程序生成的目标文件经过链接程序的处理之后才能形成一个应用程序**，也就是**可执行文件**。

### 5 setjmp和longjmp的具体意义是什么？

setjmp(envbuf)把当前的状态信息保存到envbuf中，供之后longjmp()恢复状态信息时使用，如果是直接调用setjmp(envbuf)，那么返回值为0，但是如果是由于调用longjmp()而调用setjmp()，那么返回值非0。longjmp()用于恢复由最近一次调用setjmp()时保存到envbuf的状态信息。

### 6完成递归程序，需要完成哪两个步骤？

初步建立递归程序时，首先要将**递归的公式**写出来，然后再把**递归的边界条件**写出来，返回固定值。当执行递归程序时，会将函数不断调用，从未知走向已知，**最终达到边界条件**，这就叫递推，然后从已知出发，**逐一求值回归，最终达到递归的入口处，结束回归**，这样就完成了递归程序。

### 7使用递归完成程序：给出整数m、n，求出m的n次方

```c
int m_n(int m, int n)
{
	if (n == 0)
		return 1;
	if (n == 1)
		return m;
	return m * m_n(m, n - 1);
}
```

```c
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>
#include<stdlib.h>

int go_to_m_n(int m, int n)
{
	if (m == 0 || n == 0)
		return 1;
	return go_to_m_n(m-1,n)+go_to_m_n(m,n-1);
}
int main()
{
	int x, y;
	printf("输入m和n");
	while (scanf("%d %d", &x, &y) != EOF)
	{
		printf("(0,0)到(%d,%d)有%d个方法\n",x,y, go_to_m_n(x, y));
	}
	return 0;
}
```
